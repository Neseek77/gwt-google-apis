Release Notes for @GWT_VERSION@

Added support for the Google AJAX Search API and the Google Maps API.

Known Issues

  • The MapsSample fails to load the image tiles on Safari Mac OSX.
  • The MapsSample cannot navigate between the different use cases that it wants to demonstrate on Safari Windows.
  • The Maps API requires strict HTML mode but not all of the GWT widgets will work correctly in this mode.
