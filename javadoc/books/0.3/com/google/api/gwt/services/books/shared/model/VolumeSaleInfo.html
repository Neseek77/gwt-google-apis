Interface VolumeSaleInfo

All Superinterfaces:
BaseProxy, ValueProxy
public interface VolumeSaleInfo
extends ValueProxy

Model definition for Books.VolumeSaleInfo. Any information about a volume related to the eBookstore and/or purchaseability. This information can depend on the country where the request originates from (i.e. books may not be for sale in certain countries). This class models an object which will be transmitted to or from the Books service.

Author:
Google, Inc.
