Interface Location

All Superinterfaces:
BaseProxy, ValueProxy
public interface Location
extends ValueProxy

Model definition for Latitude.Location. A Location resource identifies a user's position at a particular time. It may include metadata about the user's position, such as a venue if the location was recorded at the time of a check- in. This class models an object which will be transmitted to or from the Latitude service.
