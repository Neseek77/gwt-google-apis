Package com.google.api.gwt.services.latitude.shared Description

Support classes to facilitate interaction with v1 of the latitude service.

Lets you read and update your current location and work with your location history

To get started in your code, call GWT.create(Latitude.class)

For more information about this service, see the API Documentation
