Package com.google.api.gwt.services.calendar.shared Description

Support classes to facilitate interaction with v3 of the calendar service.

Lets you manipulate events and other calendar data.

To get started in your code, call GWT.create(Calendar.class)

For more information about this service, see the API Documentation
